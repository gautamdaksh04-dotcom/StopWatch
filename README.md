Tic-Tac-Toe Web Game

Overview

This project is a simple and interactive Tic-Tac-Toe game developed using HTML, CSS, and JavaScript. It allows two players to play the classic game directly in a web browser with a clean and responsive user interface.

Features
Two-player turn-based gameplay (X and O)
Interactive 3×3 game grid
Automatic winner detection
Display message for winner
Restart game functionality
Responsive and user-friendly design
Technologies Used
HTML – For structuring the game layout
CSS – For styling and layout design
JavaScript – For game logic and interactivity
How to Use
Open the project in a web browser
The game board will appear with 9 empty boxes
Player 1 starts the game (usually “X”)
Players take turns clicking on empty boxes
The game automatically checks for a winner
Once a player wins, a message is displayed
Click the “Restart Game” button to play again
Project Structure
index.html → Contains the game layout
styles.css → Handles design and styling
script.js → Contains game logic (player turns, winner check)
README.md → Project documentation
Future Enhancements

Conclusion

The Tic-Tac-Toe project demonstrates fundamental web development skills such as DOM manipulation, event handling, and responsive design, making it a great beginner-friendly project.
